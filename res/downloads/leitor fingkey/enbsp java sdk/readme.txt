# Tutorial para desenvolvimento de aplicacoes com NITGEN Fingkey Hamster I
# Em Java

1) Download NITGEN SDK

	SDK NITGEN para Windows 64bits (100mb)
		http://www.4shared.com/rar/smsObgYi/Nitgen_SDK_for_64.htm
	SDK NITGEN para Windows 32bits (45mb)
		http://www.4shared.com/file/ejriHf6a/Nitgen_SDK_for_32_v484.htm

	Instalar - Executar Setup.exe

2) Download eNBSP SDK para JAVA (450kb)
	http://www.4shared.com/zip/fJ2J31wi

	Extrair
	Copiar as duas DLL's(NBioBSPJNI.dll & NBioBSPISO4JNI.dll) que se encontram em "modules/windows/" para 	"C:/windows/system32"

3) Criar projeto java
	Colocar o jar NBioBSPJNI.jar que se encontra "na pasta extraida acima/lib/" no buildpath do projeto

4) Tutorial de referencia da biblioteca NBioBSPJNI.jar
	http://www.nitgen.com.br/Content/Uploads/Downloads/EN%20eNBSP%20SDK%20for%20Java%20Programmer%27s%20Guide%20DC1-0085A.pdf
